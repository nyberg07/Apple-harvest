# My Web Project

This project is a simple web application that separates HTML, CSS, and JavaScript into distinct files for better organization and maintainability.

## Project Structure

```
my-web-project
├── src
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── main.js
│   └── index.html
├── package.json
└── README.md
```

## Files Description

- **src/index.html**: The main HTML document that structures the content of the web application. It links to the CSS and JavaScript files.
- **src/css/styles.css**: Contains the styles for the web application, including layout, colors, fonts, and other visual elements.
- **src/js/main.js**: Contains the JavaScript code for the web application, handling interactivity, DOM manipulation, and any other client-side logic.
- **package.json**: The configuration file for npm, listing the dependencies and scripts for the project.

## Getting Started

To get started with this project, clone the repository and run the following command to install the necessary dependencies:

```
npm install
```

After that, you can open the `src/index.html` file in your web browser to view the application.
